피드백 화면 `/quiz/feedback/page.tsx`의 다국어 처리 패턴:

1. **언어 뱃지**: 피드백 카드 우측 상단에 "Tiếng Việt" pill 형태로 표시
2. **베트남어 설명**: 정답/오답/건너뜀 3가지 상태별로 별도 텍스트 관리
   - 정답: "Chính xác!" + 왜 맞는지 베트남어 설명
   - 오답: "Sai rồi!" + 어디서 틀렸는지, 정답이 왜 맞는지 베트남어 설명
   - 건너뜀: "Đã bỏ qua" + 건너뜀 안내
3. **접이식 한국어 해설**: 피드백 카드 내부 `button` + 조건부 렌더링. `aria-expanded` 사용.
   - 기본 닫힘 상태, 탭 시 열림/닫힘 토글
   - "📖 한국어 해설" 라벨 + 화살표 아이콘 회전 애니메이션

**Why:** 모국어(베트남어) 피드백이 주이고, 한국어 해설은 심화 학습용으로 구분.

**How to apply:** 다른 언어 추가 시 `QUESTION.feedback` 객체에 언어별 `{ title, detail }` 추가. 언어 뱃지 텍스트와 피드백 데이터 소스를 사용자 설정 언어 기준으로 교체.
