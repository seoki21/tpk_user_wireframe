# TOPIK 문제은행 앱 — CLAUDE.md

> 이 파일은 Claude Code가 프로젝트 컨텍스트를 파악하기 위한 핵심 문서입니다.

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 앱 이름 | TOPIK 문제은행 |
| 정의 | 다국어 기반 한국어능력시험(TOPIK) 학습 플랫폼 |

## 6. 유저플로우 핵심 결정사항 (v6 확정)

- **탭바 5개**: 대시보드 · 기출문제 · 연습문제 · 오답노트 · 마이페이지
- **답안 제출 제거**: 문제 풀기 → 결과 화면 직행 (자동 채점)
- **건너뛰기**: 별도 버튼 없음. "다음 문제" 클릭 시 내부적으로 건너뜀 기록
- **즉시 피드백**: 선택지 탭 즉시 정답/오답 표시 + 모국어 피드백
- **결과 후 4갈래**: 문제 해설 · 영상 강의(TBD) · 대시보드 복귀 · 다시 풀기
- **오답 노트**: 별도 탭 추가 (구현 준비 중 — 오답노트·학습관리·마이페이지 구조 검토 중)
- **학습 기록**: 마이페이지 제거 → 대시보드로 흡수

## 7. 화면별 UX 결정사항

### 대시보드
- 섹션 순서: 시험목표 → 연속학습 → 추천학습 → 학습진단
- 시험 목표: D-day 뱃지만 (전체 학습 진행률 바 제거)
- 연속 학습: "스트릭" → "연속 학습"으로 순한글 명칭 변경
- 학습 진단: 약점 분석 + 취약 유형 Top3 통합 카드 (요약 텍스트 → 태그 → 유형 리스트)
- 다국어 메뉴: 우상단 드롭다운 (🇰🇷🇺🇸🇨🇳🇯🇵🇻🇳, SVG 국기 아이콘)

### 기출문제 vs 연습문제 차이
| 항목 | 기출문제 | 연습문제 |
|------|----------|----------|
| 회차 정보 | O (제102회 TOPIK I 등) | X |
| 필터 | 등급→영역→유형 | 등급→영역→유형→난이도 |
| 실전모드 | 회차 선택 필수 | 등급·난이도(+랜덤)·문항수(미니/일반) 선택 |
| 상단 요약 칩 | 회차·등급 표시 | 등급·영역·유형 표시 |

### 유형선택풀기 (기출 · 연습 공통)
- 영역(듣기/읽기) 선택에 따라 문항유형 목록이 변경됨
- 듣기: 대화 이해, 세부 내용 파악, 주제·요지 파악, 화자 의도 파악, 적절한 응답 고르기
- 읽기: 어휘·문법, 글의 주제 파악, 글의 순서 배열, 빈칸 채우기, 세부 내용 이해, 추론하기
- 영역 미선택: 혼합 유형 표시
- 영역 전환 시 선택된 유형 초기화

### 추천학습 브리핑 (/quiz/recommend)
- 대시보드 "바로 풀기" → 추천학습 브리핑 페이지로 이동 (대시보드 UI 변경 없음)
- 3블록 구조: 최근 틀린 유형(ErrorLight) → 약점 포인트(WarningLight) → 추천 안내 + CTA(하단 고정)
- 진단(스크롤)과 행동(하단 고정) 영역을 borderTop으로 시각적 분리
- "문제 연습 시작" → 기존 /quiz 플로우로 이동

### 문제 풀기
- 진행 표시: 상단에 "N / 총N" + 진행 바만 (정답·오답·건너뜀 카운트 제거)
- 체크(플래그) 기능: 제거
- 문항 이동 바: 실전모드에서만 표시
- 피드백 언어: 사용자 설정 언어 (예: 베트남어 "Sai rồi!" / "Chính xác!")
- 강의 링크: 피드백 카드 하단에 작게 표시

### 실전모드
- 헤더: 진한 보라(#26215C) 배경 + 타이머 중앙
- 회차 선택: 최근 5개 칩(컴팩트) + `···` 아이콘 버튼 → 심플 버텀시트(전체 회차)
- 버텀시트: 드래그 핸들 + 헤더 + 회차 텍스트 목록 + 체크마크(✓) 선택 표시, 선택 시 즉시 닫힘
- 결과 버튼: "문항별 피드백 보기" (메인) + "대시보드로 이동" (보조)

### 선택풀기 결과
- 버튼 3개: 유사 문제 다시 풀기 (메인) · 다른 유형 선택 · 대시보드
- 영상 강의 버튼: 결과 화면에서 제거 (피드백 카드 내 링크로만)

### 로그인 / 회원가입
- 단일 페이지에서 탭 전환 (로그인 ↔ 회원가입, 슬라이딩 pill 인디케이터)
- 이메일 폼 상단, 소셜(Google·Apple) 아이콘 원형 버튼 하단
- FloatingInput 컴포넌트 (플로팅 라벨, 포커스 시 라벨 상승 애니메이션)
- 비밀번호 찾기: 인풋 아래 오른쪽 정렬 텍스트 링크
- /signup 접근 시 /login?tab=signup으로 리다이렉트

### 온보딩
- 언어 선택 화면에 "마이페이지 → 설정에서 언제든 변경 가능" 명시
- 목표 레벨: TOPIK I 1급 / 2급 / II 3~4급 / 5~6급 / 아직 모르겠어요
- 건너뛰기 옵션 포함

## 8. 뷰 모드 (모바일 / PC)

- 기본은 항상 모바일 (375px 폰 프레임)
- 상단 28px 다크 바(`#2D2D2D`)에 `Mobile | Desktop` 토글 (좌측 정렬)
- PC 모드: 좌측 사이드바(200px) + 콘텐츠(max 760px), 전체 max 1024px
- 선택 상태는 `localStorage`에 저장 → 페이지 이동해도 유지
- MobileFrame 컴포넌트가 모드에 따라 레이아웃 분기 (각 페이지 코드 수정 불필요)
